## O que mudou (tudo)

**Design & Identidade Visual**
- Tema tático profissional: fundo `#070B14`, accents em **ciano elétrico** `#00D4FF`, **âmbar** `#FFB800`, **vermelho de alerta** `#FF1744`
- Tipografia: `Rajdhani` (headers militares), `Share Tech Mono` (dados/IPs), `Exo 2` (texto)
- Grid animado no background, barras de progresso de risco, badges de status coloridos
- Ocultado branding do Streamlit — parece produto próprio

**Arquitetura & Engine**
- Parsing Nmap XML completo com suporte a portas abertas e serviços
- **Risk Scoring automático** (0–100) por dispositivo: desconhecido, suspeito, bloqueado, portas de risco
- 28 portas mapeadas com nível de ameaça (Telnet = CRÍTICO, RDP = ALTO, etc.)
- Classificação tripla: **Conhecido / Suspeito / Bloqueado**

**6 Abas funcionais**
| Aba | Função |
|---|---|
| ⬡ Dashboard | Métricas, tabela de ativos, distribuição de risco, vendors |
| 🖥 Dispositivos | CRUD completo, scan individual por IP, notas de operador |
| 🚨 Eventos | Log com filtros por nível, exportação CSV |
| 🔌 Portas & Serviços | Inventário de portas em toda a rede, gráfico de frequência |
| 📋 Relatórios | PDF oficial com dados da org, CSV/JSON, backup, checklist de hardening |
| ⚙ Configurações | Nome da org/unidade para o PDF, reset, status do sistema |

**Para rodar:**
```bash
mkdir ~/netwatch && cd ~/netwatch
# cole os arquivos aqui
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
